# Day 47 — Pomodoro + Todo Integrated App

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A productivity app combining a Pomodoro timer with a
todo list — tasks auto-link to Pomodoro sessions.

## How to view
Open `index.html` in your browser.

## Features
- Pomodoro timer with session tracking
- Todo list linked to timer sessions
- Mark which task you're working on
- Session history with task names
- Persistent data via localStorage
